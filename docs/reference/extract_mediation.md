# Re-export extract_mediation from medfit

Generic function to extract mediation structure from fitted models. This
function is provided by the medfit package and re-exported here for
convenience.

## Arguments

- object:

  Fitted model object

- ...:

  Additional arguments passed to methods

## Value

A
[`medfit::MediationData`](https://data-wise.github.io/medfit/reference/MediationData.html)
object

## See also

[`extract_mediation`](https://data-wise.github.io/medfit/reference/extract_mediation.html)
for full documentation
